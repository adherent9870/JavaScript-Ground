Debouncing

What is a debouncer?

Debouncing is a technique that is used to control how function is executed, it ensures that function should run only a certain time, without being event trigger again and again. example. jaisse hum search functionality ko implement krna hai toh hum agr hrr click pr api call krenge toh there will a lot of unnecessary call ho jaegi , toh inko reduce krne ke liye is technique a use krte hai jisme ek timer set ho jaega delay ke liye. 

            function debounce(fn, delay) {
            let timer;
            return function (...args) {
                     clearTimeout(timer);
                     timer = setTimeout(() => fn.apply(this, args), delay);
                };
             }

            function fetchResults(query) {
                console.log("Fetching results for:", query);
                // Imagine calling an API here
            }

            const searchBox = document.getElementById("search");
            searchBox.addEventListener("keyup", debounce((e) => {
            fetchResults(e.target.value);
            }, 500));
