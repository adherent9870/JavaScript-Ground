JavaScript for...of Loop

Complete guide with examples and interview tips

What is for...of?

The for...of statement creates a loop iterating over iterable objects (including Array, Map, Set, String, TypedArray, arguments object, etc.), invoking a custom iteration hook with statements to be executed for the value of each distinct property. 

// Basic syntax
for (variable of iterable) {
  // code block to be executed
}

Key Points:

  • Introduced in ES6 (2015)
  • Iterates over values (not keys)
  • Works with any iterable object
  • Cannot be used with objects directly
  • Cleaner syntax than traditional for loops

Basic Examples

// Iterating over an array
const fruits = ['apple', 'banana', 'orange'];
for (const fruit of fruits) {
  console.log(fruit);
}
// Output: apple, banana, orange

// Iterating over a string
const str = 'hello';
for (const char of str) {
  console.log(char);
}
// Output: h, e, l, l, o

// Iterating over a Set
const uniqueNumbers = new Set([1, 2, 3, 3, 4]);
for (const num of uniqueNumbers) {
  console.log(num);
}
// Output: 1, 2, 3, 4

Using with break and continue

const numbers = [1, 2, 3, 4, 5];
for (const num of numbers) {
  if (num === 3) break; // Exit loop when num is 3
  console.log(num);
}
// Output: 1, 2

Comparison with Other Loops

Loop Type Usage Best For Limitations
for...of Iterating over values of iterables Arrays, Strings, Maps, Sets Doesn't work with objects directly
for...in Iterating over enumerable properties Objects May iterate over prototype properties
forEach Iterating over array elements Arrays Cannot break or continue
for (traditional) General purpose looping Any situation needing index control Verbose syntax

Interview Questions

Common Questions:

  • What is the difference between for...of and for...in?
  • Can you use for...of with objects? If not, how would you make it work?
  • How does for...of handle async/await compared to forEach?
  • What are iterable protocols in JavaScript?
  • How would you iterate over a Map using for...of?

Sample Answers:

// Difference between for...of and for...in
const arr = [3, 5, 7];
arr.foo = 'bar';

for (const i in arr) {
  console.log(i); // logs "0", "1", "2", "foo"
}

for (const i of arr) {
  console.log(i); // logs 3, 5, 7
}

Advanced Usage

// Iterating over a Map
const map = new Map([['a', 1], ['b', 2]]);
for (const [key, value] of map) {
  console.log(key, value);
}
// Output: a 1, b 2

// With async/await
async function processArray(array) {
  for (const item of array) {
    await processItem(item); // Waits for each process to complete
  }
}

// Using with entries(), keys(), values()
const numbers = [10, 20, 30];
for (const [index, value] of numbers.entries()) {
  console.log(index, value);
}
// Output: 0 10, 1 20, 2 30

Making Objects Iterable

const myObject = {
  data: [1, 2, 3],
  [Symbol.iterator]() {
    let index = 0;
    return {
      next: () => {
        if (index < this.data.length) {
          return { value: this.data[index++], done: false };
        } else {
          return { done: true };
        }
      }
    };
  }
};

for (const item of myObject) {
  console.log(item); // 1, 2, 3
}
