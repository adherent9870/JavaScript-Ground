ASync and Await

what is Async/Await?

ye ek JavaScript ka feature hai jo promises ke upr bna hai jisme hum asynchronous code ko handle krte hai

Async Keyword

  • async function ko declare krne ke liye
  • ye ek promise return krta hai
  • aur function ke andr hum ek "await" keyword ka bhi istmal krte hai

Await Keyword

  • ye sirf async function ke andr hi use ho skta hai
  • ye promise ke resolve hone ka wait krta hai mtlb code pause ho jata hai jb tk kuch return nhi hota
  • agr promise reject ho jata hai toh error de dega

Notes:

  • async function hamesha promise return krta hai
  • agr function me return value hai toh wo resolve ho jayega
  • agr function me error hai toh wo reject ho jayega
  • await sirf async function ke andr hi use ho skta hai
  • await promise ke resolve hone ka wait krta hai
  • agr promise reject ho jata hai toh error de dega
