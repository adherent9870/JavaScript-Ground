Prototypical Inheritance

object properties ko inherit kr skte hai parent wle se "prototype chain ki madad se"

    • Simple Example
    •                     let animal = { eats: true,
                                   walk() {
                                    console.log("Animal is walking");
                          }
                    };

                    let rabbit ={jumps: true};

                // Inheritance: rabbit → animal
                    rabbit.__proto__ = animal;

                    console.log(rabbit.eats); // true (inherited)
                    rabbit.walk(); // "Animal is walking" (inherited)
    • When you call rabbit.eats, JS doesn’t find it inside rabbit. It follows the link (rabbit.__proto__) to animal. Finds eats there.

Object Based

  • Javascript ek object based language hai.
  • mtlb Javascript mein object ko create krne ke liye class ki zarurat nhi padti

NOTE::::

  • Prototype-based = inheritance via prototypes.
  • Class-based = inheritance via classes (sugar).
  • Object-based = focus only on creating objects (no strict class system).
