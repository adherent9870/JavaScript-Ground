Understanding call(), apply(), and bind()

In JavaScript, call, apply, and bind are methods available on functions that allow you to control the value of this and pass arguments in different ways.

call()

The call() method calls a function with a given this value and arguments provided individually.

// Syntax
function.call(thisArg, arg1, arg2, ...)

apply()

The apply() method calls a function with a given this value and arguments provided as an array.

// Syntax

bind()

The bind() method creates a new function that, when called, has its this keyword set to the provided value.

// Syntax
function.bind(thisArg, arg1, arg2, ...)

Key Differences

// call() - arguments passed individually
func.call(context, arg1, arg2);

// apply() - arguments passed as array
func.apply(context, [arg1, arg2]);

// bind() - returns a new function with bound context
const boundFunc = func.bind(context, arg1, arg2);
boundFunc();
