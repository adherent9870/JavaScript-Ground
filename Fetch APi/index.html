Fetch API kya hai?

Fetch API ek modern web API hai jo JavaScript mein network requests ko handle karne ke liye use hoti hai. Yeh asynchronous operations ko support karti hai aur promises ka use karti hai, jisse code zyada readable aur maintainable ho jata hai.\ in simple shabdon mein, Fetch API web applications ko server se data retrieve karne ya server par data bhejne mein madad karti hai. fetch API ek promise return karti hai jo response object ko resolve karti hai. Fetch API ka use karke hum GET, POST, PUT, DELETE jaise HTTP methods ke through requests bhej sakte hain. Fetch API ka use karke hum JSON, text, blobs, aur other data formats ko handle kar sakte hain. Fetch API ka use karke hum headers ko customize kar sakte hain, jaise authentication tokens ya content types. Fetch API ka use karke hum error handling kar sakte hain, jaise network errors ya server errors. Fetch API ka use karke hum CORS (Cross-Origin Resource Sharing) requests ko handle kar sakte hain. Fetch API ka use karke hum request aur response ko stream kar sakte hain, jisse large files ko efficiently handle kiya ja sakta hai.

fectch api purane XMLhttprequest se better hai kyuki ye promise based hai jisse hum async await ka use kar sakte hai aur code zyada readable ho jata hai. fetch api ka use karke hum json data ko easily handle kar sakte hai.
